# Sallie Studio — Developer Guide

This guide is for anyone building, extending, or debugging Sallie Studio.

---

## 1. Developer Console

Open: **Developer Console**

You can run:

- PowerShell
- Python
- CMD
- Docker
- Node
- Environment checks

---

## 2. Built-in Commands

env
pythonpath
dockerlogs
backendlogs
frontendlogs

---

## 3. Logs

All logs live in:

logs/

Includes:

- backend.log
- frontend.log
- docker.log
- studio.log

---

## 4. Scripts

All scripts live in:

Scripts/

You can modify them freely — the app will pick up changes automatically.

---

## 5. Themes

Themes live in:

Themes/

You can create your own theme by adding a new XAML file and referencing it in App.xaml.
