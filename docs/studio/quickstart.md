# Sallie Studio — Quickstart

This is the fastest way to get up and running.

---

## 1. Launch Sallie Studio

Open it from the Start Menu.
If this is your first time, the Setup Wizard will guide you through:

- Repo detection
- Backend detection
- Docker detection
- Port scanning
- Profile creation

---

## 2. Start Everything

Click:

**Start All**

This launches:

- Docker containers
- Backend
- Frontend
- Watchers

You’ll see live status updates on the Dashboard.

---

## 3. Check Health

Click:

**Health Check**

You’ll get a full diagnostic across:

- Backend
- Docker
- Frontend
- Ports
- Node processes

---

## 4. Use the Developer Console

Open:

**Developer Console**

You can run:

- PowerShell commands
- Python commands
- Docker commands
- Environment checks
- Log viewers

---

## 5. Switch Profiles

Use the Settings page to switch between:

- Dev
- QA
- Demo

Each profile has its own ports, repo root, and behavior.

---

## 6. Minimize to Tray

Close the window — Sallie Studio keeps running.
Right-click the tray icon for quick actions:

- Start All
- Stop All
- Health Check
- Open Studio
